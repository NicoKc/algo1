## Ejercicio 2

- a) wp(a:= a+1, a >= 0)
    ~~~
    ≡ Ax1
    ≡ { def(a+1) ∧L (a+1 >= 0) }
    ≡ { True ∧L ( a >= -1 )}
    ≡ { a >= -1 }
    ~~~

- b) wp(a:= a/b, a >= 0)
    ~~~
    ≡ Ax1
    ≡ { def(a/b) ∧L (a/b >= 0)}
    ≡ { (b != 0) ∧L ((a >= 0 ∧ > 0) ∨ (a <= 0 ∧ < 0))}
    ≡ { ((a >= 0 ∧ > 0) ∨ (a <= 0 ∧ < 0)) }
    ~~~

- c) wp(a:= A[i], a >= 0)
    ~~~
    ≡ Ax1
    ≡ { def(A[i]) ∧L (A[i] >= 0) }
    ≡ { (0 <= i <= |A|) ∧L (A[i] >= 0) }
    ~~~

- d) wp(a:= b*b, a >= 0)
    ~~~
    ≡ Ax1
    ≡ { def(b*b) ∧L (b*b >= 0) }
    ≡ { True ∧L (b*b >= 0) }
    ≡ { (b*b >= 0) }
    ≡ True
    ~~~

- e) wp(b:= b+1, a >= 0)
    ~~~
    ≡ Ax1
    ≡ { def(b+1) ∧L (b >= -1) }
    ≡ { b >= -1 }
    ~~~
